# Stage 1 課題：HTML/CSS基礎

## 課題一覧

### 課題1: 名刺サイト（meishi.html）
自分の名刺をイメージしたページを作る。

**要件:**
- 名前・肩書き（例：WEBデザイナー見習い）・一言メッセージを表示
- Flexboxでレイアウトを組む
- `:hover` を使ったインタラクションを1つ以上
- 背景色・フォントを工夫する

**NG:**
- インラインスタイル（`style=""`）は使わない
- CSSは必ず外部ファイル（style.css）に書く

---

### 課題2: コピーコーディング（copy-coding.html）
好きなサイトのファーストビューを見た目だけコピーする。

**ルール:**
- コードをコピーしてはいけない、目で見て手で書く
- 完全一致を目指さなくていい。「それっぽく見える」がゴール
- どのサイトをコピーしたか `README` に書いておく

---

### 課題3: カードコンポーネント（card.html）
商品カードを3枚並べるページを作る。

**要件:**
- カード1枚に: 画像・タイトル・説明文・ボタンが入る
- 3枚をFlexboxで横並びにする
- `:hover` でカードが少し浮き上がる（`transform: translateY`）
- スマホ（480px以下）では縦並びになる（メディアクエリ必須）
