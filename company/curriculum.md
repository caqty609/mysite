# Megu Web Studio — カリキュラム全体像

**目標**: WEB制作会社に就職できる、またはフリーランスで案件獲得できるレベルへ  
**想定期間**: 6〜8ヶ月（週15〜20時間学習）

---

## Stage 1：HTML/CSS 基礎固め（4〜6週間）

**ゴール**: セマンティックなHTMLとCSSの基礎を完全に理解する

### 学習内容
- [ ] HTMLセマンティクス（header/main/section/article/footer）
- [ ] CSSセレクタ・詳細度・継承
- [ ] Flexbox（横並び・縦並び・中央揃え・折り返し）
- [ ] CSS Grid（2次元レイアウト）
- [ ] 擬似クラス・擬似要素（:hover, ::before, ::after）
- [ ] CSS変数（カスタムプロパティ）
- [ ] Gitの基本操作（add / commit / push / branch）

### 課題プロジェクト: `projects/01_html-css-basics/`
1. **名刺サイト** — 自分の名刺をHTMLとCSSで再現する
2. **コピーコーディング** — 既存のサイトを見ながら見た目を真似る（Google/Apple等）
3. **カードコンポーネント** — 商品カード・ブログカードをFlexboxで作る

### クリア条件
- [ ] CSSを見て「なぜこう書いているか」を口頭で説明できる
- [ ] Flexboxだけで3カラムレイアウトが作れる
- [ ] GitHubにpushできる

## Stage 2：レスポンシブ + LP制作（4〜6週間）

**ゴール**: スマホ対応のランディングページを一人で作れる

### 学習内容

- [ ] Google Fonts・アイコン（Font Awesome）活用

2. **バナー練習** — 600x200 / 300x250 / 728x90 のWebバナーを各5種類制作

### クリア条件

## Stage 3：バナー練習アプリ + JavaScript基礎（4〜6週間）

**ゴール**: JavaScriptで動きのあるUIが作れる

### 学習内容
- [ ] JavaScriptの基本文法（変数・関数・条件分岐・ループ）
- [ ] DOM操作（querySelector・addEventListener）
- [ ] イベント処理（クリック・入力・スクロール）
- [ ] 非同期処理（setTimeout・fetch・async/await）
- [ ] JSONの読み書き
- [ ] LocalStorage

### 課題プロジェクト: `projects/03_banner-app/`
**バナー練習アプリ** — バナーのサイズ・テキスト・色を動的に変えられるプレビューツール
- バナーサイズ選択（ドロップダウン）
- テキスト入力でリアルタイム反映
- 背景色・文字色の変更
- 「いいね」機能（LocalStorage保存）

### クリア条件
- [ ] バナー練習アプリが完成し、GitHubに公開されている
- [ ] fetchを使って外部APIからデータを取得できる
- [ ] コードを見せながら「ここで何をしているか」を説明できる

## 就職後も見据えて身につけておくべきこと

| スキル | 重要度 | 備考 |
|--------|--------|------|
| Git/GitHub | ★★★★★ | 絶対必須。PRとレビューの文化を学ぶ |
| Figmaからのコーディング | ★★★★★ | 現場では必ずデザインカンプがある |
| レスポンシブ | ★★★★★ | できて当然のレベルで求められる |
| WordPress | ★★★★☆ | 中小制作会社の案件の半数以上はWP |
| JavaScript | ★★★★☆ | 動きをつけられないコーダーは弱い |
| React/Vue | ★★★☆☆ | あれば差別化になる |
| コミュニケーション | ★★★★★ | 技術と同じくらい重要。質問力・報連相 |
